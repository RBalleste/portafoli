Qué es html?

HTML es el llenguatge que s'utilitza per al desenvolupament de pàgines d'internet.

Esta constituit per elements que els navegador interpreten. Hi han elements per ficar imatges sobre una pàgina, hipervincles, llistes, taules...

Per a poder creas una pàgina de HTML sols es necesari un editor de text i un navegador.


Estructura interna de HTML

Les instruccions estan tancades per els caracters < i >


L'estuctura bàsica d'una pàgina HTML es:

<html> <head> <title></title> </head> <body> Cos de la pàgina </body> </html>
