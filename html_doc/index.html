HTML Documentación

Introducción

HTML és el llenguatge que s'utilitza per al desenvolupament de pàgines d'internet. Esta constituit per elements que els navegador interpreten. Hi han elements per ficar imatges sobre una pàgina, hipervincles, llistes, taules... Per a poder creas una pàgina de HTML sols es necesari un editor de text i un navegador.

Estructura interna de HTML Les instruccions estan tancades per els caracters <i >

L'estuctura bàsica d'una pàgina HTML es:

<html> <head> <title></title> </head> <body> Cos de la pàgina </body> </html>

Instruccions bàsiques: salt de linea, Paràgraf, tituls i Enfasis...
El salt de linea es fa amb <br> El que es vol ficar com a Paràgraf amb <p> </p>. Els següents cassos igual canviant la p per les altres instruccions.

Llistes

Exemple de llista desordenada i subllista ordenada, amb ul, ol i amb li

<ul> <li> Exemple 1</li> <li> Exemple 2</li> <ol> <li> Subllista 1</li> <li> Subllista 2</li> </ol> <li> Exemple 3</li> </ul> <br>


  • Exemple 1
  • Exemple 2
    1. Subllista 1
    2. Subllista 2
  • Exemple 3

El elemento ul define una lista desordenada.
El elemento contiene uno o más elementos li que especifican los ítems de la lista.
Por el contrario, el elemento ol define una lista ordenada.
El elemento contiene uno o más elementos li que especifican los ítems de la lista como una secuencia ordenada.
