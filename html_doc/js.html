Conceptes de algoritme i programació

Algoritme son els passos a seguir per resoldre un problema.

Programa es un conjunt de intruccions que enten el ordinador per a realitzar una activitat

Llenguatge de programació son el conjunt de instruccions que son interpretades per la computadora per a realitzar operacions

Que es Javascript?

És un llenguatge de seqüència que et permet crear contingut d'actualització dinàmica, controlar multimèdia, animar imatges i pràcticament tota la resta. (Està bé, no tot, però és sorprenent el que pots aconseguir amb unes poques línies de codi JavaScript).

No es pot executar fora de un navegador


El programa amb JavaScript a de anar entre la marca script i inicialitzada la propietat type amb la cadena text/javascript.

Es sensible a majuscules i minuscules

El més básic seria:

<html> <head> </head> <body> <script type="text/javascript"> document.write("Hola mon"); </script> </body> </html>

Variables

Les més bàsiques son:

  • Valors sencers (199, 234, etc)
  • Valors reals (1.23, 2,50 etc)
  • Cadenes de caracters ("Juan","Compres","llistat", etc)
  • Valors lògics (true,false)

Entrada per el teclat

Un exemple seria:>


<html> <head> </head> <body> <script type="text/javascript"> var nom; var edat; nom=prompt("ingresi el seu nom:"); edat=prompt("ingresi la seva edat"); document.write("Hola"); document.write(nom); document.write("tens"); document.write(edat); document.write("anys."); </script> </body> </html>

Estructures secuencials i condicionals de programació

Per a que JS agafi el continguts de les variables com a enters i no com a cadenes de caracters em de utiñitzar la funció parseInt, per exemple:


var suma=parseInt(valor1)+parseInt(valor2); var producto=parseInt(valor1)*parseInt(valor2);

Condicional simple

if (nota>=4) { document.write(nom+"esta aprobat amb una nota de "+nota); }

Condicional compost

Tindria la segúent estructura:


if(<condició>) { <instrucio(ns)> } else { <instrucio(ns)> }

Condicional anidat

Seria:

if (pro>=7) { document.write("promocionat"); } else { if(pro>=4) { document.write("regular"); } else{ document.write("Suspes"); } }
